# zh-kiwi-hub

## 项目简介

zh-kiwi-hub 是一个用于归档和发布独立 HTML 页面的仓库。这里存放的页面通常为单一功能或特定用途，不绑定任何特定域名或网站。您可以在此浏览、查看或引用这些页面，但请注意，每个页面均独立维护，其内容与风格可能各有不同。

## 目录说明

```
.
├── README.md          # 本文件
├── pages/             # 存放所有 HTML 页面文件（按需归入子目录）
│   ├── example/       # 示例子目录
│   └── ...
└── assets/            # 页面共用的资源文件（如 CSS、JS、图片等，可选）
```

- `pages/`：所有独立 HTML 页面均放置于此目录下，可根据用途或主题进一步划分子目录。
- `assets/`：如页面间共用样式、脚本或媒体资源，可统一存放在此，但多数页面可能自带资源以避免依赖。

## 页面归档说明

- 每个 HTML 页面均为独立归档，不依赖仓库内其他文件（除明确引用的共用资源外）。
- 页面可能包含自包含的样式与脚本，或引用外部资源（如 CDN 库），引用来源请自行核实。
- 归档的页面可能包含实验性、临时性或历史性内容，不代表长期维护或更新承诺。
- 部分页面可能随时间移除或替换，恕不另行通知。

## 维护说明

- 本仓库由个人维护，更新频率不固定。
- 欢迎通过 Issues 提出建议或报告问题，但无法保证及时处理。
- 不接受 Pull Request 用于添加新页面或修改现有页面内容（除非是明显的错误修复）。
- 页面内容可能来源于个人项目、学习记录或临时需求，恕不提供详细背景说明。

## 许可

本仓库中每个 HTML 页面均视为独立作品，其权利归属与许可情况请以页面内声明为准。如页面内未声明，则默认保留所有权利。
